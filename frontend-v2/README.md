# EduAssist V2 (Managed Rebuild)

Phase 1 focuses on core flows: role selection, login, and role dashboards.

## Setup
```bash
npm install
```

## Configure API
Create `.env` and set:
```
EXPO_PUBLIC_API_URL=http://<your-local-ip>:8000
```

## Run
```bash
npx expo start --clear
```

## Phase Plan
1. Core flows (current)
2. Student and Guardian feature screens
3. Lecturer, HoD, Finance, Records modules
4. Notifications, voice, and chat enhancements
